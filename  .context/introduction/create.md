功能编排是指通过功能模块进行高度封装，并辅以更加直观的业务逻辑组织视图，以帮助用户快速构建所需要的应用，以及更结构化的查看及管理智能体的功能。

本文将以「**景区助理**」应用创建为例，介绍如何快速构建一个功能编排应用。

## 功能编排结构说明
功能编排的整体结构分为三个核心板块，智能体执行时会按以下顺序依次执行：

| **模块** | **说明** | **是否必填** |
| :---: | --- | :---: |
| **前置判断** | 配置智能体在功能最开始需要进行的业务判断，如用户身份验证、权限检查等 | 否 |
| **快速决策** | 预先配置的FAQ信息，在固定问答内容下提升智能体回复的准确率和反应速度 | 否 |
| **意图识别** | 智能体的核心能力模块，大模型会根据用户输入自动判断用户意图并调用对应技能 | 是 |


## 创建功能编排应用
### 1. 创建应用
登录 **百宝箱企业版**，单击 **新建应用**，选择 **对话** - **功能编排** 构建方式，完成应用名称、应用简介、所属行业等基本信息填写，单击 **新建**。

![](https://cdn.nlark.com/yuque/0/2025/png/47021694/1753972206293-df718b67-9b2b-417a-a6c8-63e9df2c6639.png)

### 2. 配置前置判断（可选）
创建后进入 **整体视图 **页面

![](https://cdn.nlark.com/yuque/0/2025/png/47021694/1753972036396-137e2078-8fa6-488e-93d9-fc8506a4c92c.png)

**前置判断 **模块用于配置智能体在功能执行最开始需要进行的业务判断，如用户身份验证、业务权限检查等。该模块为非必填，若不配置则直接进入 **快速决策 **模块。

#### 2.1 选择前置判断模块
点击 **前置判断** 模块，开始配置业务判断逻辑。

![](https://cdn.nlark.com/yuque/0/2025/png/47021694/1753973764663-70b8ab4c-b971-4902-96c3-e036f6b0aaf8.png)

#### 2.2 填写判断场景逻辑
点击前置判断后，会弹出侧边栏配置界面，填写对应内容：

+ **技能名称**：为该前置判断技能命名
+ **标签**：自定义标签，便于创建者在整体视图中快速识别，同时也会帮助模型帮助快速了解该模块功能
+ **调用技能**：选择对应的进行业务逻辑判断的插件
+ **核心功能示意图**：可自行上传功能示意图，便于在整体视图中，快速点击查看

> 提示：带有 * 的为必填项，调用场景描述写得越清晰，模型的调用准确性就越高。
>

![](https://cdn.nlark.com/yuque/0/2025/png/47021694/1753973201753-208aa8df-bf22-42f4-b1d9-6979fbc5b657.png)

#### 2.3 技能调用配置
+ 在调用技能模块中，点击 **添加插件**，选择需要的业务插件。

![](https://cdn.nlark.com/yuque/0/2025/png/47021694/1753973868164-db768a0d-78ad-410a-9d7a-722bb929637c.png)

![](https://cdn.nlark.com/yuque/0/2025/png/47021694/1753972036479-a1cbdc08-160a-425b-9d9e-0909fe4684af.png)

+ 选择插件后，需要配置插件的输入/输出参数：

![](https://cdn.nlark.com/yuque/0/2025/png/47021694/1753973387024-8d9f1db0-bbf5-4f79-ba40-cfe5bf357ac1.png)

![](https://cdn.nlark.com/yuque/0/2025/png/47021694/1753972037615-72356f3e-91f6-4c75-90c4-a277070171f4.png)

+ 配置完插件参数后，点击 **配置结果**，可根据不同的出参结果，配置多个智能体相应的执行方式。

![](https://cdn.nlark.com/yuque/0/2025/png/47021694/1753974142155-eac3c3f4-2703-42b7-9708-6612455cb5c0.png)

![](https://cdn.nlark.com/yuque/0/2025/png/47021694/1753972038068-4506c807-077f-4c61-a73d-ed1e1aca3c13.png)

![](https://cdn.nlark.com/yuque/0/2025/png/47021694/1753973076894-80e5d04b-682e-4d52-b53b-10f5927bbbae.png)

#### 2.4 保存前置判断
配置完成后，点击 **确认** 保存前置判断技能。

![](https://cdn.nlark.com/yuque/0/2025/png/47021694/1753972038105-8b0b9be5-af02-432d-be61-c7bbfdd27a89.png)

保存成功后，可以看到已配置的前置判断模块。同一智能体中可以配置多个前置判断逻辑。

![](https://cdn.nlark.com/yuque/0/2025/png/47021694/1753972038370-fc595b7b-9329-4be4-b191-1c024e277e7a.png)

### 3. 配置快速决策（可选）
快速决策模块本质上是预先配置的FAQ信息，目的是在固定问答内容下提升智能体回复的准确率和反应速度。

#### 3.1 快速决策模块配置
+ **模块创建**：点击 **快速决策** 模块开始配置。该模块为非必填。

![](https://cdn.nlark.com/yuque/0/2025/png/47021694/1753972039161-280b89a5-32ea-4bd9-b2e6-147cac61542b.png)

+ **信息填写**：在弹出的配置界面中，确认或修改相关字段。
    - **功能名称**：可自定义名称
    - **调用场景**：告诉主智能体，在什么场景下会调用该模块。系统已默认调用场景描述，可自行选择是否进行修改
    - **调用技能**：固定为知识库中的基础客服，如需修改内容，可点击 **知识库-基础客服 **文档进行内容修改。

![](https://cdn.nlark.com/yuque/0/2025/png/47021694/1753972039325-11733066-95ea-4d4e-8ea5-063d60f9c8b0.png)

+ **功能保存**：点击 **确认添加** 后，快速决策模块配置完成。

![](https://cdn.nlark.com/yuque/0/2025/png/47021694/1753974595402-34902e71-014d-40ae-8089-a6a9871f4cfd.png)![](https://cdn.nlark.com/yuque/0/2025/png/47021694/1753974622342-9204b050-5757-4c56-9061-8cfa1a92ec58.png)

#### 3.2 管理FAQ内容
如需修改FAQ内容，可点击 **知识库** 进入管理界面。

![](https://cdn.nlark.com/yuque/0/2025/png/47021694/1753972039621-b7688cc1-9963-4b2c-a86b-2279e9867c79.png)

在服务管理的客服FAQ中，可以进行固定FAQ的配置和管理。

![](https://cdn.nlark.com/yuque/0/2025/png/47021694/1753972039813-ec54d01d-ec02-42bf-9c70-1b0fdcfb4e72.png)

### 4. 配置意图识别（必填）
意图识别是功能编排的核心模块，智能体的主要能力都在该模块中配置。大模型会根据用户输入的内容自动判断用户意图，并调用对应的技能。

#### 4.1 选择意图识别模块
点击 **意图识别** 模块，该模块为必填项。

![](https://cdn.nlark.com/yuque/0/2025/png/47021694/1753972040329-4f6a94bd-c663-4074-92cf-6d10d26629d7.png)

#### 4.2 填写意图信息
在侧边栏中填写具体的技能信息：

+ **技能名称**：为该意图技能命名
+ **调用场景描述**：详细描述在用户有什么需求时可以调用该技能，**最好可以针对典型场景进行举例说明**。调用场景描述越清晰，模型调用准确性越高
+ **标签**：添加标签帮助快速了解该模块功能，同时提升大模型调用准确率
+ **调用技能**：选择对应的功能插件

![](https://cdn.nlark.com/yuque/0/2025/png/47021694/1753972040579-5414e689-1565-4186-b921-e4c5bea6ee61.png)

#### 4.3 配置插件参数
选择插件后，需要对插件的输入参数进行配置。

![](https://cdn.nlark.com/yuque/0/2025/png/47021694/1753972041053-a76e1a4c-fff9-4b98-85ab-e94e4deee1e0.png)

#### 4.4 保存意图技能
点击 **确认** 后，成功添加该意图下的技能。

![](https://cdn.nlark.com/yuque/0/2025/png/47021694/1753972041035-0860d8c1-4fea-4095-9c8a-0c5b9748499a.png)

用户可以添加多个技能，在该框架下，智能体会自动根据用户的输入内容及上下文，自动识别用户意图并调用对应的能力。

![](https://cdn.nlark.com/yuque/0/2025/png/47021694/1753972041326-b1dde263-436b-484e-a437-29be7c7f681f.png)

### 5. 全局参数配置
点击 **全局参数配置** 可以进行业务相关参数以及环境参数的配置。

![](https://cdn.nlark.com/yuque/0/2025/png/47021694/1753972041514-7cd6d02c-65d1-4a21-84d7-b06e1a594462.png)

根据具体技能插件的需要，可以将一些与业务逻辑相关的、对全局有影响的参数配置在此处，供各技能调用。

![](https://cdn.nlark.com/yuque/0/2025/png/47021694/1753972042055-c48da78b-9916-4b65-adf4-0452fcd1bc0f.png)

环境参数介绍，参考 [系统及环境参数](https://alipaytbox.yuque.com/sxs0ba/huntb8/tpzqsb1zaiblggmy) 文档

![](https://cdn.nlark.com/yuque/0/2025/png/47021694/1753972042244-45669c40-b287-4184-9c57-b1a413a44462.png)

### 6. 对话配置
按照相关字段配置对应的对话框展示效果，具体说明可查看[详细功能说明](https://alipaytbox.yuque.com/sxs0ba/huntb8/gqkhmccq343ldyad)文档。

![](https://cdn.nlark.com/yuque/0/2025/png/47021694/1753972042937-2e8d0474-4636-430a-a7c1-54900dc3b5af.png)

![](https://cdn.nlark.com/yuque/0/2025/png/47021694/1753972042957-fe354586-58f6-4053-9e37-f567149429be.png)

### 7、测试应用
配置完成后，可以点击右上角的 **运行** 按钮进行效果调试。

![](https://cdn.nlark.com/yuque/0/2025/png/47021694/1753972042909-e88ede23-eb19-41f3-991b-b6127becf200.png)

运行测试时，左侧会显示对应模块是否调用成功，以及若失败时的具体失败原因，便于开发者进行调试和优化。

![](https://cdn.nlark.com/yuque/0/2025/png/47021694/1753972043783-a85ab1f1-688a-4060-9457-0a1d781eec0a.png)



最后，调试完成功能完善后，可点击发布进行保存，并进一步进行渠道上架。

## 注意事项
+ 意图识别模块为必填项，前置判断和快速决策模块可根据业务需要选择性配置
+ 调用场景描述的准确性直接影响大模型的调用准确率，建议详细描述使用场景
+ 全局参数配置中的参数会影响所有技能的执行，请谨慎配置
+ 建议在正式发布前充分测试各个模块的功能，确保业务逻辑正确
